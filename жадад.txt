Вот адаптированный текст под описание встроенного калькулятора Windows с сохранением исходной структуры и стиля.

---

1 НАЗНАЧЕНИЕ ПРОГРАММЫ

1.1 Назначение программы
Программа «КАЛЬКУЛЯТОР WINDOWS» предназначена для выполнения базовых, инженерных, программистских и статистических вычислений, а также для конвертации величин и расчётов с датами с целью предоставления пользователю быстрого и удобного инструмента для решения повседневных и профессиональных математических задач.

Программа состоит из нескольких функциональных блоков:

· Пользовательский интерфейс: область ввода/вывода данных, панель с цифровыми и функциональными кнопками, меню выбора режима работы.
· Вычислительное ядро: набор алгоритмов, выполняющих арифметические, тригонометрические, статистические и другие виды вычислений.
· Конвертер величин: модуль для перевода значений между различными единицами измерения (длина, вес, объём, температура и др.).
· Калькулятор дат: модуль для вычисления разницы между датами и добавления/вычитания дней от заданной даты.

1.2 Возможности программы
Программа позволяет:

· выполнять базовые арифметические операции (сложение, вычитание, умножение, деление);
· производить инженерные вычисления (тригонометрические, логарифмические, степенные функции, работа с константами);
· выполнять расчёты в системах счисления с основаниями 2, 8, 10, 16 с поддержкой логических операций (режим "Программист");
· проводить статистические вычисления (режим "Статистика");
· конвертировать значения между различными единицами измерения;
· вычислять разницу между двумя датами или определять дату путём прибавления/вычитания дней;
· сохранять историю вычислений в памяти программы;
· использовать клавиатуру для ввода чисел и операций.

1.3 Основные характеристики программы
В программе реализовано 4 основных режима работы ("Стандартный", "Научный", "Программист", "Вычисление даты"), каждый из которых предоставляет специализированный набор математических функций и инструментов, адаптированных под конкретный тип задач.

1.4. Ограничения, накладываемые на область применения программы
Программа является настольным приложением для выполнения математических и конвертационных расчётов и не предназначена для символьных вычислений, построения графиков функций или решения сложных систем уравнений. Программа поставляется в составе операционных систем семейства Microsoft Windows, её автономная установка или работа под управлением других операционных систем не предусмотрены.

2 УСЛОВИЯ ПРИМЕНЕНИЯ

2.1 Минимальный состав технических средств
В состав технических средств должен входить IBM-совместимый персональный компьютер (ПЭВМ) или планшет, включающий в себя:

· процессор с тактовой частотой, ГГц - 0.5, не менее;
· операционную систему Windows 7, Windows 8, Windows 10, Windows 11;
· оперативную память объемом, Мб - 128, не менее;
· устройство ввода (клавиатура, мышь или сенсорный экран).

2.2 Минимальный состав программных средств
Системные программные средства, используемые программой, должны быть представлены одной из поддерживаемых версий операционной системы Microsoft Windows (Windows 7 и выше). Для функционирования программы не требуется установка дополнительного программного обеспечения.

3 ОПИСАНИЕ ЗАДАЧИ

3.1. Определение задачи
Основная задача, решаемая программой – предоставление пользователю доступного, точного и быстрого инструмента для выполнения широкого спектра математических операций, конвертации единиц измерения и расчётов с датами в рамках повседневной, учебной и профессиональной деятельности.

3.2. Методы решения задачи
Калькулятор Windows представляет собой программное обеспечение, реализующее математические алгоритмы и предоставляет следующие возможности:

· интерактивный ввод данных через графический интерфейс или клавиатуру;
· немедленное выполнение вычислений по принципу "ввод-операция-ввод-равно";
· хранение промежуточных результатов в памяти (кнопки M+, M-, MR, MC);
· переключение между режимами для доступа к различным наборам функций;
· отображение полной истории выполненных операций в текущей сессии.
  Данное ПО состоит из:
· графического пользовательского интерфейса (окно с дисплеем, кнопками и меню);
· вычислительного модуля, содержащего библиотеки математических функций;
· модуля конвертации единиц измерения;
· модуля расчёта операций с датами.

4 ВХОДНЫЕ И ВЫХОДНЫЕ ДАННЫЕ

4.1 Сведения о входных данных
Входные данные представляют собой последовательность действий пользователя, включающую:

1. Числа: Вводятся с цифровых кнопок интерфейса или клавиатуры. Поддерживаются целые, дробные (с разделителем-точкой) и отрицательные числа.
2. Математические операции: Выбираются нажатием на соответствующие кнопки (+, -, *, /, sin, log и т.д.) или вводом с клавиатуры.
3. Управляющие команды: Очистка ввода (CE), очистка всего (C), использование памяти (MS, MR, M+, M-), вычисление результата (=).
4. Выбор режима: Установка одного из четырёх режимов работы через меню "Вид".
5. Данные для конвертации: Выбор исходной и целевой единицы измерения, ввод числового значения.
6. Даты: Выбор дат из графического календаря или ввод с клавиатуры.

4.2 Сведения о выходных данных
Выходными данными является числовой или текстовый результат, отображаемый на основном дисплее программы. В зависимости от режима это может быть:

· результат арифметического или научного вычисления;
· число в выбранной системе счисления (двоичной, восьмеричной, шестнадцатеричной);
· результат конвертации единиц измерения;
· разница между датами в днях или результирующая дата.
  Дополнительно, в интерфейсе программы отображается история последних выполненных операций.
